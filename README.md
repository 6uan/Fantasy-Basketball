# FANTASY BASKETBALL


> [!NOTE]
> Created a Virtual Environment by Running the Following Commands:



 ``` 
$ python3 -m venv .venv
$ . .venv/bin/activate
 ```

- After we have our .venv install all required dependencies 

 ``` 
$ pip install -r requirements.txt
$ npm install
 ```

- Watch for TailwindCSS changes

 ``` 
$ npm run watch:css
 ```

- Start Flask Server

 ``` 
$ python main.py
 ```
