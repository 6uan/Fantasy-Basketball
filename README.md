# FANTASY BASKETBALL


> [!NOTE]
> Created a Virtual Environment by Running the Following Commands:



 ``` 
$ python3 -m venv .venv
$ . .venv/bin/activate
 ```

- After we have our .venv install all required dependencies 

 ``` 
$ pip install -r requirements.txt

 ```

- Start Flask Server

 ``` 
$ flask --app main run
 ```
